Changelog for Blendermada addon

0.9.0 to 0.9.1

  • Added favorites functionality. Now you can get API key, place it in settings and see materials, marked as your favorites in separate category

0.3.0 to 0.9.0

  • Improved description field.
  • Added icons for better UI.
  • Fixed trouble with importin material to object without slots.

0.2.2 to 0.3.0

  • Rearranged widgets to make more compact view.
  • Added setting to switch between big and small preview.
  • Added setting to change cache path. Change it if you have problems with preview/import.

0.2.1 to 0.2.2

  • Fixed Blender 2.76 import path issue.

0.1.3 to 0.2.1

  • Added material preview

0.1.2 to 0.1.3

  • Change server API. For now client shows only materials for active render engine

0.1.1 to 0.1.2

  • Added support of Blender 2.72 new append method

0.1.0 to 0.1.1

  • Duplication control. If a material with name like imported already exists, importing canceled.
  • Imported material replaces material in active slot.

Client is still under active development. If you find a bug, of have another issue, you can find this project on GitHub or write me directly - ozzyrov@gmail.com
